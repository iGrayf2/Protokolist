# Предобработка аудио

Начиная с версии `0.3.0`, Protokolist перед распознаванием по умолчанию готовит стабильный WAV-файл для Faster-Whisper.

Исходная запись не изменяется. Подготовленный файл сохраняется в папку встречи:

```text
output/<дата>_<имя_записи>/artifacts/prepared_audio.wav
```

## Что делает ffmpeg

Перед Whisper выполняется конвертация:

```text
mono WAV, 16 kHz, pcm_s16le
```

Это делает вход для модели более предсказуемым, особенно если исходник пришел из телефона, диктофона, мессенджера или видеофайла.

## Обычный запуск

```bash
./scripts/transcribe_file_ubuntu.sh input/meeting.mp3
```

В этом режиме Whisper получает уже подготовленный файл `prepared_audio.wav`.

## Отключение предобработки

Если нужно передать исходный файл напрямую в Whisper:

```bash
./scripts/transcribe_file_ubuntu.sh input/meeting.mp3 --no-preprocess
```

Этот режим полезен для сравнения качества или отладки.

## Следующий шаг
