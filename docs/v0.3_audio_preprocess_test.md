# Проверка v0.3 audio preprocessing

Минимальная проверка после обновления `main`.

## 1. Обновить репозиторий на Ubuntu

```bash
cd ~/Protokolist
git pull
```

## 2. Проверить окружение

```bash
PYTHONPATH=src .venv/bin/python -m protokolist.cli doctor
```

## 3. Быстрая проверка на короткой записи

```bash
./scripts/transcribe_file_ubuntu.sh input/test_short.mp3 --model tiny
```

Ожидаемый результат в логе:

```text
[2/7] Предобработка аудио через ffmpeg
  ✔ .../artifacts/prepared_audio.wav
[3/7] Распознавание аудио через Faster-Whisper
```

## 4. Проверить файлы

В новой папке встречи должны появиться:

```text
artifacts/prepared_audio.wav
artifacts/meeting.raw.json
artifacts/meeting.raw.txt
artifacts/meeting.cleaned.txt
artifacts/meeting.cleaned.srt
artifacts/meeting.cleaned.docx
CHATGPT_PACKAGE/
*_chatgpt_package.zip
```

## 5. Проверка режима без предобработки

```bash
./scripts/transcribe_file_ubuntu.sh input/test_short.mp3 --model tiny --no-preprocess
```

Ожидаемый результат:

```text
[2/7] Предобработка аудио через ffmpeg
  ! Предобработка отключена, Whisper получит исходный файл
```
